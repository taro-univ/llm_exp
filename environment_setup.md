# 研究環境（RTX 5070 + PostgreSQL）再開ガイド

このドキュメントは、デスクトップPC（SOTARO_PC）を再起動した際に、ノートPCから研究環境を復旧させるための最短手順をまとめたものです。

## 1. デスクトップPC（SOTARO_PC）側の準備
物理的な電源を入れた後、以下のサービスがバックグラウンドで起動していることを確認してください。

* **Tailscale**: VPN接続（IP: 100.xxx.xxx.xxx）がアクティブであること。
* **Docker Desktop**: 起動していること（これが動いていないとコンテナが立ち上がりません）。
* **ログイン**: Windowsへのユーザーログインを済ませておいてください。

## 2. ノートPCからの接続
ノートPCの VS Code を開き、以下の手順でデスクトップの Linux 領域へアクセスします。

1.  左下の **[青いアイコン（><）]** をクリック。
2.  **「Connect to Host...」** を選択し、デスクトップPCのIPアドレスを選択。
3.  接続後、**`File > Open Folder`** から以下のパスを開く。
    * パス: `/home/shona/llm_exp`

## 3. 開発コンテナ（Dev Container）の起動
フォルダが開くと、VS Code が自動的に構成を検知します。

1.  右下に表示される **「Reopen in Container」** ボタンをクリック。
2.  **初回のみ**: 構成を変更した場合はビルドが走りますが、通常は数秒でコンテナ内に入れます。
3.  ターミナルが `root@...:/workspace#` になれば成功です。
